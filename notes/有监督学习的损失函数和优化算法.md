# 凸函数
    凸函数曲面任意两点的连线，其上的任意一点都在这两点之间曲面的上方
    L(a * x + (1 - a) * y) <= a * L(x) + (1 - a) * L(y)

# 交叉熵
    L = -sum(yi * log(pre_yi)) , 每个样本的损失， 因为yi只在某一类上取1，所以是在最大化在该类上的概率

# 随机梯度下降
    应用场景： 梯度下降会计算所有的数据，计算量大
    方法： 每次只计算一个随即样本的梯度，进行更新
